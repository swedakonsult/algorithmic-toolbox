# Problem Description

*Problem*. Given two digits a and b, find a+b.

*Input format*. The first line of the input contains two integers a and b (separated by a space).

*Constraints*. 0≤a,b≤9.

*Output format*. Output a+b.

## Sample 1

Input:

```
1
3 2
```

Output:

```
1
5
```

## Sample 2

Input:

```
1
7 9
```

Output:

```
1
16
```
